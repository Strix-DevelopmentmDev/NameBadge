# Name Badge

A fully optimized and synced **prop-based name badge script** for FiveM.  
This script allows players to display their **character name, rank, or job title** using a physical badge prop attached to their character.

Perfect for **law enforcement, EMS, DOJ, government, corporate jobs, and immersive professional roleplay**.

---

## Features

- Prop-based badge system
- Displays **firstname**, **lastname**, **rank**, and **job title**
- Fully synced with nearby players
- Supports **ESX**, **QB-Core**, **QBOX**, and **custom frameworks**
- Easy configuration
- Easy localization
- Job restrictions
- Placement mode for custom badge positioning
- SQL saving for badge offsets/positions
- Custom object support
- Optimized for performance
